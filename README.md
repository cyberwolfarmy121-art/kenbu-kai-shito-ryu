# Ken Bu Kai Shito Ryu - Karate Teaching Website

A luxurious and attractive karate teaching website featuring video tutorials, membership system, and comprehensive admin panel.

## Features

### Video Training
- **Basics** - Fundamental karate techniques
- **Kata** - Traditional forms and patterns
- **Kumite** - Sparring techniques and strategies

### Membership System
- Three membership tiers: Basic ($29), Premium ($59), Master ($99)
- Videos are locked based on membership level
- Membership key verification system

### Admin Panel
Access with membership key: `ADMIN2024`

- Manage members (view, add, delete)
- Upgrade memberships
- Change website logo
- Add/remove masters and champions
- Edit contact information

### Profile System
- My Achievements
- My Grade (with belt display)
- Membership Details

## Getting Started

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Use membership key `ADMIN2024` to access admin panel
